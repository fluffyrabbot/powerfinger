# Contributing to PowerFinger

Want to collaborate or contribute? Add **fluffyrabbit** on Discord.

We're looking for:

- **EE** — ESP32-C3/S3, BLE HID, flex PCB layout, low-power design
- **ME / Industrial Design** — Parametric ring shell CAD, 3D printing for body-worn devices
- **Firmware** — ESP-IDF, FreeRTOS, NimBLE, HID report descriptors
- **Accessibility testers** — People who use alternative input devices daily

See [docs/OUTREACH.md](docs/OUTREACH.md) for the full list of communities and
organizations we're connecting with, and [docs/PROTOTYPE-SPEC.md](docs/PROTOTYPE-SPEC.md)
for the current build spec.
